マクロ変数の展開

v1.2で追加された機能

マクロ変数を鍵括弧[]で囲んで文字列に展開する際、以下の指定ができるようになった

(1) フォーマット指定

以下のように変数名の前にコロンで区切ってフォーマットを指定する


[%fmt:var]

v1.2より前では、例えば


tip> @ x=exp(10)

tip> fmt %10.2e
tip> prn [x]
  2.20e+04

のようにfmtコマンドで指定していたが、v.1.2以後は、


tip> fmt reset
tip> prn default format: [x], specified format: [%10.2e:x]
default format: 22026.465795, specified format:   2.20e+04

のように、1行で複数のフォーマットを併用できるようになった

(2) 数式を評価して展開


[expr(var)]

鍵括弧[]の中にマクロ変数を含んだ数式を記述し計算結果を文字列に展開できるようになった

例：


tip> @ pi=acos(-1)

tip> prn [2*pi]
6.2831853072

ちなみに、2*[pi]は、piが文字列に展開された後に計算されるが、 [2*pi]は、先に計算した後に結果を文字列に展開する

例：


tip> calc 2*[pi]
[2*3.1415926536] -> [6.2831853072]

tip> calc [2*pi]
[6.2831853072] -> [6.2831853072]

数式評価はフォーマット指定と同時に使用できる

例：


tip> prn [%.2f:2*pi]
6.28

フォーマットの区切り、及び、数式にはスペースを入れてもよい

例：


tip> prn [%.2f : 2 * pi]
6.28
