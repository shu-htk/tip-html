tip - tiny interactive plot

Tip is a command-line based data plotting tool. It is written in C++ and depends on the library of PLPLOT, and GNU readline. Basically it works on Linux or other Unix-like OS. Although it has fewer functions than existing plotting tools, it just aims ease of use over versatility.

Features

  • plot the curve in 2D and 3D
  • plot the histogram in 2D and 3D
  • plot the FFT power spectrum
  • data fitting to the curve and the histogram
  • data sorting, cutting
  • control (for,do,while,end,if,elif,else,fi)
  • execute macro file
  • save to pdf or png file
  • EPICS CA access (optional)
